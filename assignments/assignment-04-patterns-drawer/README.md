# Assignment 04 — Patterns Drawer

## Overview

This assignment focused on using **loops** to draw repeated visual patterns. The program creates different graphical designs such as stars, triangular grids, random circles, and spiral-like paths.

## What I Built

- Star pattern
- Triangular grid
- Random circles
- Spiral path challenge

## My Approach

I used loops to control repeated drawing actions instead of manually writing every shape. For patterns that changed position or size, I updated variables inside the loop so each new shape appeared in the correct place.

## Key Takeaways

- Loops are useful when a task follows a repeated pattern.
- Small changes to variables inside a loop can create complex visual results.
- Drawing patterns requires both programming logic and coordinate reasoning.

## Looking Back

If I revisited this assignment today, I would:

- Reduce repeated drawing code.
- Use named constants for important measurements.
- Make the pattern settings easier to adjust.
- Add comments explaining the loop logic for each pattern.
