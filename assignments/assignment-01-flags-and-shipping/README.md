# Assignment 01 — Shipping Calculator & Flag Drawer

## Overview

My first COMP102 assignment consisted of two independent Java programs. The **Shipping Calculator** models a real-world shipping pricing system, while the **Flag Drawer** recreates national flags using Java graphics and proportional drawing techniques.

## What I Built

- Shipping Calculator Core
- Shipping Calculator Completion
- Shipping Calculator Challenge
- Flag Drawer

## My Approach

For the Shipping Calculator, I broke the pricing model into smaller calculations before combining them into the final result. For the Flag Drawer, I used proportional measurements instead of fixed coordinates so the flags could scale correctly for different sizes.

## Key Takeaways

- Programming is easier when problems are broken into smaller steps.
- Using proportions makes graphical programs more flexible.
- Organising calculations before coding makes debugging much easier.

## Looking Back

If I revisited this assignment today, I would:

- Improve variable naming consistency.
- Extract repeated calculations into helper methods.
- Add input validation.
- Include screenshots of the graphical output.
