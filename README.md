## Portfolio Philosophy

This repository documents my journey learning Java throughout COMP102 at Victoria University of Wellington.

Rather than simply uploading assignment solutions, I use each project to record how my approach to problem solving evolved as I learned new programming concepts.

Each assignment includes:

- **Overview** — what the project was about.
- **What I Built** — the completed programs and challenge extensions.
- **My Approach** — how I designed and solved the problem.
- **Key Takeaways** — the most important lessons I gained.
- **Looking Back** — how I would improve the project with my current knowledge.

My goal is to document progress, not perfection. As I continue learning software engineering and cybersecurity, this repository serves as a record of how my thinking has developed over time.
