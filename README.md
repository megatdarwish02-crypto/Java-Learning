# Java Learning Journey

This repository documents my Java programming journey through COMP102 at Victoria University of Wellington.

I am using this repository as more than a code dump. My goal is to show how I learned Java fundamentals, how my thinking developed across assignments, and how I moved from simple procedural programs into object-oriented programming.

## Why This Repository Exists

When I first started Java, many concepts were not instantly obvious to me. Things like object references, `==` vs `.equals()`, ArrayLists, file reading, and object interaction only started to make sense after drawing diagrams and repeatedly asking *why* the code behaved a certain way.

This repository is my attempt to preserve that learning process properly.

## Java Concepts I Focused On

- Writing small, readable methods
- Using variables and constants properly
- Understanding parameters and return values
- Reading files with `Scanner`
- Understanding the difference between primitives and objects
- Understanding references, heap memory, and stack variables
- Using arrays and ArrayLists
- Building programs with multiple classes
- Thinking about program state in games and GUI programs

## Portfolio Philosophy

A lot of beginner repositories only show final code. I want this repository to show both the code and the thinking behind it.

For each assignment, I am documenting:

1. **Problem** — what the assignment was trying to solve.
2. **Approach** — how I broke it into Java logic.
3. **Concepts** — what Java ideas were involved.
4. **Reflection** — what I understand now that I did not understand before.
